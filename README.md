#アプリケーションの概要

ユーザー投稿型東京ランダム案内アプリケーションです。
東京のオススメグルメ、観光場所を位置情報付きで共有できます
お気に入り登録するとマイページからも確認いただけます。


#アプリケーション機能一覧

・ユーザー登録/ログイン機能
・SNSログイン認証機能
・ユーザー投稿、編集機能
・アクセス制限
・部分検索機能
・Google Map機能
・ページネーション機能
・お気に入り機能
